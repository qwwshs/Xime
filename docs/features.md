# 功能指南

## 语音转文本

语音转文本功能支持通过麦克风进行实时语音识别：

- **双引擎支持** - 支持在线 API（阿里百炼）和本地离线模型
- **实时识别** - 边说边识别，结果实时显示
- **自动标点** - 根据语句特征自动添加标点符号
- **离线可用** - 本地模型无需网络，隐私安全

详细说明请参考 [语音转文本](features/speech-to-text.md)。

## 剪贴板管理

剪贴板功能自动保存复制的内容：

- **剪贴板标签** - 查看所有复制历史
- **快捷发送标签** - 查看收藏的内容
- **置顶** - 长按可置顶重要内容
- **添加到快捷发送** - 将内容收藏到快捷发送
- **删除** - 删除单条记录
- **清空** - 清空所有剪贴板历史

## 快捷发送

快捷发送功能允许用户收藏常用的文本内容，以便快速输入：

- **添加内容** - 在剪贴板标签中点击星形图标添加到快捷发送
- **快速发送** - 在快捷发送标签中点击内容即可插入
- **管理收藏** - 删除不需要的收藏内容
- **永久保存** - 收藏的内容会一直保留，不会被新复制的内容覆盖

## 键盘调节

键盘调节功能允许用户根据个人习惯调整键盘的外观和尺寸：

- **高度调整** - 拖动键盘上方的控制条来改变键盘高度
- **默认高度** - 一键恢复到默认高度设置
- **高度范围** - 键盘高度限制在屏幕高度的1/3到1/2之间
- **确认保存** - 调整完成后点击确认按钮保存设置

## 表情

表情功能提供丰富的表情符号和自定义表情插件：

- **内置表情** - 包含笑脸、手势、动物、食物等分类的表情符号
- **表情插件** - 支持扩展表情插件，添加颜文字、表情包等
- **分类浏览** - 按类别浏览表情，便于查找
- **快速插入** - 点击表情即可插入到输入框中

## 部署方案

部署方案功能用于重新加载和应用输入方案配置：

- **重载配置** - 重新加载当前的输入方案配置
- **应用更改** - 在切换输入方案或修改设置后需要重载配置
- **等待时间** - 重载过程需要几秒钟，请耐心等待

## 输入方案

输入方案管理允许用户选择不同的输入方法：

- **方案列表** - 显示所有可用的输入方案
- **切换方案** - 点击选择不同的输入方案
- **当前方案** - 显示当前激活的输入方案
- **重载生效** - 切换方案后需要重载配置才能生效
