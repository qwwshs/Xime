# 方案市场 + 依赖补齐

给曦码加入「方案市场」：从官方索引浏览/搜索远程 Rime 方案，一键安装并**自动补齐依赖**（反查词典、词频 essay、符号 symbols 等），再一键部署生效。

相比"纯列表 + 下载"，本实现的核心增量是 **依赖补齐引擎**：很多方案（仓颉、速成、笔画…）的反查、连打依赖额外文件（`luna_pinyin`、`essay`、`symbols`），缺了就会"装上却报缺 / 连打打不出词"。本功能在安装时按方案声明的 `dependencies` 递归补齐，并对 app/上游已自带的基础文件做"本地优先"判断，不重复下载。

## 数据源

- 索引：官方端点 `https://index.ximei.me`（代理 `ximeiorg/xime-index`），失败回退 jsDelivr / raw.githubusercontent。
- 方案/依赖包：各 version 的 `downloadUrl`，直连上游（GitHub Releases/Archive）。
- 依赖要生效，需 `xime-index` 索引为相关方案登记 `dependencies` 并注册库包条目（`luna_pinyin`/`symbols`/`essay`）——见该仓库 PR。

## 分层

```
UI(SchemaMarketScreen/ViewModel)
  └─ XimeIndexSource  取数(端点+镜像兜底)+ 安装编排
       ├─ XimeIndexParser/Models   纯函数解析(可单测)
       ├─ SchemaManager.importFromUrl  下载+校验+安全解压
       └─ RimeDependencyResolver   按 dependencies 补齐(本地优先)
```
解析/版本/兼容性是纯函数（充分单测）；网络与 Android 副作用集中在取数层。

## 逐 commit 说明（为什么需要 + 作用）

| commit | 改了什么 | 为什么需要 / 作用 |
|---|---|---|
| `feat(schema): SchemaManager 启用写 schema_list + 安全导入` | F1 把启用方案写进 `default.yaml` 的 `schema_list`；导入加 sha256 校验、zip-slip 防穿越、GitHub 归档剥壳目录、tar.gz gunzip、保护 `default.yaml`/`*.custom.*` | librime 只编译 `schema_list` 里的方案，不写就"启用了也编不出"；安全导入避免校验缺失/路径穿越，并修"无 `.schema.yaml` 的包(如 rime-essay)文件落进子目录 rime 读不到"导致 essay/连打失效 |
| `feat(deps): RimeDependencyResolver 依赖补齐(本地优先)` | 按 `dependencies` 递归下载补齐反查/词库；本地已具备则跳过；如实返回仍缺项 | 让仓颉/速成等带反查、连打完整可用；本地优先避免重复下载 app/上游已带的 `symbols`/`default` |
| `feat(market): xime-index 数据模型 + 纯解析层` | kaml 模型 + `parseIndex`/`parseScheme`/`resolveRepoPath`/`isCompatible` 等纯函数 | 把索引解析做成可 100% 单测的纯逻辑，与网络解耦 |
| `feat(market): XimeIndexSource 取数 + 安装编排` | 从 `index.ximei.me` 取索引(jsDelivr/raw 兜底)，按版本 sha256 安装，装后调补齐引擎 | 大陆可达性 + 安装后自动补齐的编排入口 |
| `feat(market): 方案市场 UI + 导航/入口 + 持久已安装 + 一键部署` | SchemaMarketScreen/ViewModel(搜索、兼容性拦截、安装态、一键部署)；主设置入口 + 导航；已安装改持久记录；ON_RESUME 刷新走 IO | 用户可浏览/搜索/安装/部署；"已安装"用持久记录而非文件存在性(方案可能仅作为依赖落盘) |
| `fix(dict): 词库查看器跟随 import_tables` | 查看器递归合并 `import_tables` 的词条 | quick5/cangjie5 的词在导入表里、主词典本体为空，原查看器显示为空 |
| `docs(market): 本说明` | 本文档 | 逐 commit 记录设计与必要性，便于 review/PR |

## 验证

- `./gradlew :app:testDebugUnitTest`：解析/补齐/导入/词库等纯逻辑单测全绿。
- 端到端：方案市场装仓颉/速成 → 自动补齐 luna_pinyin/symbols/essay → 一键部署 → 反查与连打可用。
